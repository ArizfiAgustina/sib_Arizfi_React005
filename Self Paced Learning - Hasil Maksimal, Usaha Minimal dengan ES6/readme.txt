Deklarasi variable baru yaitu Let dan Const.
- Let hanya bisa diubah di value nya saja, berfungsi di scope yang sama.
- Const menciptakan satu variable yang tidak bisa diubah, bersifat read only.

- Function Arrow dapat dilihat dengan =>, tulisan dapat lebih ringkas
- defaul parameter : membantu menulis kode lebih sedikit, membuat function lebih fleksibel
